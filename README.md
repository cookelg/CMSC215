# CMSC 215 Intermediate programming
### Learning objectives:
- Design and implement object-oriented programs using features like classes, objects, and inheritance.
- Use object-oriented programming approach to simplify a complex programming problem.
- Demonstrate the use of exception handling.
- Create graphical user interfaces including user-interface controls, event handlers and animations.
- Apply the concepts and techniques of recursive programming
- Demonstrate the use of generic classes, interfaces, and methods.
- Report the design, implementation, testing and debugging process through proper program documentation.
